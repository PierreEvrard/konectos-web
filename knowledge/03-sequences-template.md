# 03 — Séquences multi-touch (cross-platform)

Documente ici les séquences actives (LinkedIn → WhatsApp → Instagram, etc.).
Chaque bloc = une séquence nommée.

## Format recommandé

### Nom de la séquence

- **Déclencheur** : [À CONFIGURER]
- **Étapes** :
  1. [Plateforme] — [Action] — [Délai]
  2. …
- **Statut** : brouillon | active | en pause
- **Mesure succès** : [À CONFIGURER]

---

## Exemple (template)

### Nouveau lead LinkedIn → WhatsApp

- **Déclencheur** : contact a commenté un post ciblé
- **Étapes** :
  1. LinkedIn — icebreaker (message `to`) — J0
  2. LinkedIn — relance — J+5
  3. WhatsApp — 1er message (si numéro obtenu) — J+7
  4. WhatsApp — relance — J+10
- **Statut** : brouillon
- **Mesure succès** : % réponse, % clic sur `goalLink`, % passage au CTA
